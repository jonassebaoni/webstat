This is a Meteor based project, with react on the client side

# Requirement:
- npm
- meteor

To install meteor :
```
curl https://install.meteor.com | /bin/sh;
```
To install SyncedCron for meteor :
```
meteor add percolate:synced-cron
```

# Install
```
npm install; meteor
```

# Enjoy
browse to http://localhost:3000
